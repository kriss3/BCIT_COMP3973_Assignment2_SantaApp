Very Marry Christmas, Santa!

Welcome to your new Sinple Page Application to manage Kids.

This App was prepared by Krzysztof Szczurowski for Assignment 2 COMP 3973

Below is a list of techologies used:

  • ASP.NET Core 2.1 and C# for cross-platform server-side code
  • Angular 6 and TypeScript for client-side code, in VS2017
  • Bootstrap for layout and styling
  • Bing Maps to display childs's location

Useful details:

  • Navigation. Click Childrento return to this page.
  • Accounts:. User Name: santa with default password (specified in requirements) as ADMI rights. Any other account is of type CHILD.
  • Notes: Application has number of useful alert notification and JSON objects displayed to monitor current state of variables and forms. In PROD system this would be removed and sub with logging framework.

User can LOG OUT via Children tab => left pane. User can SEE current logged used in the top part of Children pane.
